1️⃣ What is the difference between var, let, and const?
answer:- `var` is function-scoped and can be redeclared and updated. It is hoisted to the top of its scope. `let` is block-scoped and can be updated but not redeclared. It is also hoisted but not initialized. `const` is block-scoped and cannot be updated or redeclared. It must be initialized at the time of declaration.

2️⃣ What is the spread operator (...)?

answer:- The spread operator (`...`) allows an iterable such as an array or string to be expanded in places where zero or more arguments (for function calls) or elements (for array literals) are expected. It can also be used to copy objects and arrays, merge arrays, and pass elements of an array as individual arguments to a function.

3️⃣ What is the difference between map(), filter(), and forEach()?

answer:- `map()` creates a new array by applying a function to each element of the original array. It returns the new array. `filter()` creates a new array with all elements that pass a test implemented by the provided function. It also returns the new array. `forEach()` executes a provided function once for each array element but does not return a new array; it returns undefined.
